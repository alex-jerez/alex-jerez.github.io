Requirements

  • A terminal environment running Python 2.7+

  • A text editor or IDE (Sublime Text, PyCharm, etc)

  • If you’ll be connecting to our server, you will need a terminal based editor like vim, nano, emacs

Connect to our Linux Server

You may connect to our dedicated Linux server if you would prefer not to create a python environment on your computer.

Create your Environment and Activate it

mkvirtualenv pythonworkshop

Your terminal should have (mkvirtualenv) before the dollar sign.
