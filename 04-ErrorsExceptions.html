Exceptions

Exceptions happen when your code is syntactically correct, but something went wrong while the code was being executed.


# AttributeError:

>>> w = "word"
>>> w
'word'
>>> w.append('hello')
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AttributeError: 'str' object has no attribute 'append'


# NameError:

>>> w = "word"
>>> print r
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
NameError: name 'r' is not defined

# ZeroDivisionError:

>>> print 5/0
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ZeroDivisionError: integer division or modulo by zero

Using try and finally to work around errors.

>>> phrase = "Hello"
>>> try:
...     phrase.append(" World")
... except AttributeError:
...     phrase += " World"
... 
>>> print phrase
Hello World
