03 Classes, Functions and Methods

Functions

a function is a reusable user-defined action.

def sayHello():
    print "Hello"

def saysomething(x):
    print x

>>> def sayHello():
...     print "Hello"
... 
>>> def saysomething(x):
...     print x
... 
>>> sayHello()
Hello
>>> saysomething("HEY")
HEY
>>>

*args and **kwargs

adding *args and **kwargs to your function lets your function accept any argument. This is helpful for when you don’t know what variables you will be expecting.

*args catches all positional arguments

**kwargs catches all the keyword arguments.

def printeverything(*args, **kwargs):
    for item in args:
        print item
    for key, value in kwargs.items():
        print key, value

This function printeverything() can now accept as many arguments and keyword arguments as you can throw at it.

def printeverything(*args, **kwargs):
    for item in args:
        print item
    for key, value in kwargs.items():
        print key, value

Classes & Methods

class TransportationVehicle(object):
     def __init__(self):
        self.wheels = None
        self.color = None
        self.brand = None

class Car(TransportationVehicle):
    def __init__(self, color, year, make):
        self.color = color
        self.year = year
        self.make = make

    def __repr__(self):
        return "this car is a %s %s %s" % (self.color,
                self.year,
                self.make)

    def about(self):
        print "this car is a %s %s %s" % (self.color,
                self.year,
                self.make)



car_one = Car("red", 2005, "Toyota")
car_two = Car("black", 2001, "Honda")

Static Methods

a static method is a function inside of your class that doesn’t depend on the class object. By adding a @staticmethod decorator to your function, it becomes a static method.

    @staticmethod
    def honk():
        print "HONK!"

Class Methods

a class method is another type of function inside of your class that has to take a class object as the first argument.

Note that this is different from accepting an instance of the class. This means that the function can return a class type that contains all the original functions and attributes.

A common use case for class methods are for initializing classes under arbitrary conditions.

@classmethod
def honkAndSay(cls, word):
    cls.honk()
    print word
